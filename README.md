# Telegram Media Downloader Bot

A powerful Telegram bot built with Python that leverages **yt-dlp** and **FFmpeg** to download and process videos or audio from YouTube and other platforms directly into your chat.

---

## 🚀 Getting Started

### Prerequisites
Before you begin, ensure you have the following installed on your system (e.g., Ubuntu, macOS, or Windows):
* **Python 3.10+**
* **FFmpeg**: Essential for merging video/audio streams and audio extraction.
* **Telegram Bot Token**: Obtain this from [@BotFather](https://t.me/BotFather).
